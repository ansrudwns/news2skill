# TurboQuant and EDEN Lineage

## Concept
TurboQuant is not an isolated breakthrough but mathematically derived from the 1-bit quantization schema "DRIVE" (NeurIPS 2021) and the generalized "EDEN". Understanding this lineage allows us to implement TurboQuant without blindly trusting its default scaling factors.
